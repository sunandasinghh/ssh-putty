This project demonstrates how Secure Shell (SSH) is used to securely access a remote Linux system using PuTTY from a Windows machine.
The project focuses on understanding SSH fundamentals, secure remote login, and basic Linux command execution over an encrypted channel.


 Objectives of the project:
*Understand the SSH protocol
*Learn how PuTTY works as an SSH client
*Establish a secure remote connection
*Execute Linux commands remotely
*Learn the security benefits of SSH

Tools used:
*PuTTY (SSH Client)
*Linux Server (VM / Remote server)
*Windows OS
*SSH Protocol (Port 22)
